# Product Overview: X-Engineer

## Purpose & Value Proposition
X-Engineer is a bilingual (JA/EN) static web platform and technical content hub that:
1. Serves as a **talent bridge** connecting global engineers with Japanese SIer enterprises
2. Acts as a **cross-posting engine** for technical articles to Qiita, Zenn, and GitHub Pages
3. Provides **practical AWS/DevOps knowledge** to address Japan's IT talent shortage

## Key Features
- Bilingual UI (Japanese/English toggle) with language-aware content sections
- Interactive candidate registration and corporate inquiry forms with validation
- Markdown-rendered technical documentation hub with tabbed navigation
- Dynamic grid layout (4-card top, unified middle doc hub, full-width bottom)
- GitHub Pages deployment via GitHub Actions CI/CD
- Zenn CLI integration for article management

## Target Users
- **Candidates**: Global engineers (JLPT N2+) seeking Japanese enterprise roles; AWS bootcamp participants
- **Enterprises (SIer)**: Japanese IT companies needing cloud-native talent for AWS, DevOps, IaC, FinOps
- **Readers**: Engineers consuming AWS/cloud technical articles on Qiita, Zenn, and the site itself

## Core Technical Topics Covered
- DR / Backup strategies (RTO/RPO minimization)
- Cloud migration (on-prem to AWS)
- Cost optimization & FinOps
- Multi-account management (AWS Control Tower)
- Observability (CloudWatch, OpenTelemetry, distributed tracing)
- AI-assisted Lambda development
- AWS Amplify full-stack patterns
- IaC with Terraform (pipelines, landing zones)
- Compliance automation (AWS Config, Security Hub)
- XOps pipelines (DevOps + SecOps + GitOps unified CI/CD)
- Agentic DevOps (AI-driven issue resolution, sprint automation)

## Live Site
