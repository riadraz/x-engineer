# AWS IoT Greengrass V2 入門

## Greengrass V2 とは？

AWS IoT Greengrass V2 は、エッジデバイス上で  
- Lambda コンポーネント  
- コンテナ  
- ML 推論  
- デバイス管理  
を安全に実行するための **エッジランタイム**です。

## 主な特徴

- コンポーネントベースの柔軟なデプロイ  
- ローカルメッセージング  
- OTA 更新  
- Fleet Provisioning と組み合わせた大規模展開

## ユースケース

- キオスク端末  
- 工場設備のエッジ制御  
- 車載ゲートウェイ  
- 小売店舗のセンサー集約
- スマート農業
- スマートシティ
